To use you need to:
1. Download python 3.9.13
2. You need to download the data from https://www.kaggle.com/datasets/zippyz/cats-and-dogs-breeds-classification-oxford-dataset
and then add it to the folder called data_images_v2

This project contains two machine learning models that can identify 12 cat breeds
In the dropdown menu you can select the model you want to use to do a prediction
To do the prediction you need to add an image in the right part of the application and press the button under the image
